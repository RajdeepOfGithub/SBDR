**Hardware Environment:** Mac Pro M3 Pro | 24GB Unified Memory | macOS Sequoia/Sonoma 

Technical Execution Strategy

- **Data Processing:** Use **Pandas** (or Polars for speed) to merge the Lending Club dataset with synthetic chat logs.
- **NLP Engine:** Fine-tune **FinBERT** using the `mps` device in PyTorch to analyze "Financial Anxiety" from text.
- **Behavioral Engine:** Train a **Bidirectional LSTM** to identify declining spending patterns in the 6-month payment history.
- **Integration Layer:** Use a **Random Forest** or **XGBoost** model to combine the NLP scores and LSTM outputs into 4 recovery tiers. 

Project Significance

- **Innovation:** Moves from traditional "risk scoring" to "empathy scoring."
- **Feasibility:** Fully executable locally on M3 Pro hardware without needing expensive cloud clusters.
- **Impact:** Reduces customer defaults while maintaining brand loyalty through automated compassionate outreach.
